# PTA資料候補一覧

このフォルダは、PTA運営適正化に関係する資料候補を分類したものです。PDF本体の確定掲載ではなく、どの資料がどの論点に関係するかを整理するための補助データです。

## 収録ファイル

- `source_candidates.csv`：資料候補一覧
- `source_candidates.json`：同内容のJSON版
- `site_import_plan.md`：サイト内での使いどころ

## 注意

資料を本文で引用する場合は、リンク先の本文、発出主体、発出日、現行性、重複の有無を確認してください。研究資料、報道、自治体マニュアル、国通知は資料の性格が異なるため、同じ重みで扱わないようにします。


## Stage18 追加

`page_expansion_map.csv` / `page_expansion_map.json` は、資料候補を各論点ページの本文にどう展開したかを示す対応表です。資料を確定根拠として断定するためではなく、確認質問・照会項目・開示請求項目へ変換するための作業用整理です。
